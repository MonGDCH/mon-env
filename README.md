# mon-env

基于PHP的配置管理工具, 实现以下功能:

* 多种配置文件格式支持, 支持array、json、ini、xml、yaml等类型
* 支持自定义扩展配置文件解析驱动
* 目录、文件批量配置解析

## 安装

```bash
composer require mongdch/mon-env
```
